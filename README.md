# Kengomori.com

Personal developer portfolio of **Kengo Mori (杜健豪)**.

Built with Next.js and deployed on Vercel.

---

## Overview

Minimal, fast portfolio site to present my work as a software engineer in Tokyo.

This site will evolve into a central hub for:
- Projects
- Technical writing
- Professional contact
- Future products (including SaaS and e-commerce)

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Vercel

---

## Pages

- `/` → Home
- `/about` → About me
- `/projects` → Projects
- `/contact` → Contact

---

## Goals

- Clean and minimal design
- Fast performance
- Mobile-first layout
- Easy to extend (blog / products later)

---

## Future Expansion

- Blog (engineering + learning notes)
- Case studies for projects
- API integrations
- Subdomain services (e.g. shop.kengomori.com)
